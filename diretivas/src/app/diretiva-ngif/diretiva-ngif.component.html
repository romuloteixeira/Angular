Angular CLI - Pré processador de CSS

Novo projeto

  • ng new nome-projeto --style=sass
  • ng new nome-projeto --style=less
  • ng new nome-projeto --style=stylus

Projeto existente - Changing default CSS

  • ng set defaults.styleExt scss
  • ng set defaults.styleExt less
  • ng set defaults.styleExt styl

Comandos qualidade de código - lint, test e e2e

  • ng lint
  • ng test
  • ng e2e
